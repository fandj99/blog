## 安装

```bash
git clone https://github.com/fandj99/blog.git
&
cd blog
```

## 下载依赖&启动
```shell
# npm
npm install
&
npm run dev

# yarn
yarn install
&
yarn dev
```


```shell
# npm
npm run build

# yarn
yarn build

```
